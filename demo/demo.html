Default behavior

Integers only, allows steps, no min/max values, no real time change event

Min/Max & Decimal places

3 decimal places with minimum value = -10 and maximum value = 10

Configurable steps value

Step up increments by 2 and step down decrements by 3

Disable steps

Only allows typing directly

Real-time change event with "quiet millis" = 500

When steps occur, or when the value is adjusted to fit the min/max values, a change event will automatically be triggered.

"Quiet millis" delays the change event by waiting the given amount after the user stops changing the value.
Try keeping the up arrow pressed and then stop and wait a bit.
